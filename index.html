Updated Thurs. February 15, 2018 - 15:27

Interviewer view

The calls vs claims report displays details of individual interviewer performance and casework by survey. Select the desired view, choose necessary filters and press the Generate report button. Definitions are found in the glossary at the bottom of the page.

All fields marked with an asterisk ( *) are mandatory.

Glossary

Terms
Responses
The number of resolving attempts with submitted responses that meet the Minimum Content Criteria and are not Out of Scope.
Refusals
The number of attempts with an outcome code in the 500 series.
No contacts
The number of attempts with an outcome code in the 100 and 200 series.
Other outcomes
The number of attempts with an outcome code that does not fall into the Response, Refusals, and No Contacts categories.
Interview length
Time spent inside the electronic questionnaire within an attempt.
System time
Total time spent on an attempt. Includes time contacting respondent, performing the interview, and applying an outcome code.
Calls
The number of attempts made via telephone during this period.
Visits
The number of attempts made via a field visit during this period.
Unique cases touched
The number of unique cases for which attempts have been made during this period.
Surveys
The number of surveys for which an attempt has been made during this period.
Total off system time
The total amount of time where an interviewer was not making an attempt during this period, measured between the first and last attempt of the day.
Access method
How the case was delivered to the interviewer for this attempt. Possible values are:
  • Assigned: A
  • Self-Assigned: SA
  • Call Scheduler: CS
  • Call Scheduler - Appointment: CSA
Attempted via
Whether the attempt was made via a telephone call ( ), or a field visit ( ).
